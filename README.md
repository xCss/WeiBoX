# WeiBox -- 微博图床客户端

- [x] 支持点选/拖拽方式上传图片至新浪微博
- [x] 支持批量上传
- [x] 可选择返回的图片尺寸
- [x] 可生成`图片链接`、`HTML`、`UBB`和`MarkDown`四种格式
- [x] 支持返回`https`安全协议的图片地址

## 🚀 Features
- [ ] 历史记录同步
- [ ] 浏览和删除历史记录
